What is this tool?▼▲

This tool is an advanced machine learning system designed to aid medical professionals in classifying various types of blood cells. It's meant to streamline the process, improve accuracy, and facilitate early detection of potential hematological issues.

What data will the system use?▼▲

The system will analyze images of blood samples obtained through quantitative phase imaging. This allows observing cells without staining. The main cell types are red blood cells, white blood cells, platelets, aggregates, and out of focus cells.

How does the pipeline work? ▼▲

It starts with segmenting and extracting cells from the images. Then features are extracted, and the cells are labeled. Finally, models are trained to classify cell types. Active learning helps select useful samples for labeling.

What capabilities does the frontend provide?▼▲

The frontend includes an interface for labeling cells, training models, making predictions, and correcting errors. It visualizes results and explains model decisions. The goal is an intuitive, responsive UI to present analyses and aid diagnosis. Creativity is encouraged for visuals, interpretations, and more.
